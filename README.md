# Rayyan Abhram Personal Website

Personal portfolio site with resume, gallery, and projects overview. Built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Structure

```
/app
  layout.tsx
  page.tsx
  projects/page.tsx
  resume/page.tsx

/components
  Navigation.tsx
  Hero.tsx
  About.tsx
  Resume.tsx
  Gallery.tsx
  Projects.tsx
  Contact.tsx
  Footer.tsx
```

## Update content

- Edit copy in the component files under `/components`.
- Update `CONTACT_EMAIL` in `components/Contact.tsx`.
- Replace gallery cards with real images in `components/Gallery.tsx`.
- Replace `public/resume.pdf` with your actual resume PDF.
